Installation

1. Get Steam

Download Steam at http://store.steampowered.com/about/.

2. Get Source SDK Base 2007

Open steam://run/218 in your browser to start the Source SDK download.
Alternatively go to your Steam Library and click the drop down menu next to the search box under the word LIBRARY. It probably says ALL GAMES for you. Select Tools and look for Source SDK Base 2007 in the list. Right-click and select Install Game....

3. Get IOSoccer

Download the game here and extract the game files to your sourcemods folder (default path is C:\Program Files (x86)\Steam\steamapps\sourcemods).

4. Restart Steam

The game will show up in your library after Steam has been restarted.

Troubleshooting

• The game doesn't start

Run Source SDK Base 2007 once.
Let it load until you see the main menu, then click Quit. Some people need to do this to be able to run IOSoccer, so just run it to be on the safe side.
